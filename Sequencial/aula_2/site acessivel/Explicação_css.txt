body { ... }:

Este seletor body aplica estilos ao corpo do documento HTML, ou seja, à área principal onde todo o conteúdo é exibido. Aqui estão definidos estilos como a família de fontes, tamanho da fonte, espaçamento entre linhas, margem, padding e cor de fundo.
header { ... }:

Este seletor header aplica estilos ao cabeçalho do site. Define a cor de fundo, cor do texto e espaçamento interno do cabeçalho.
h1 { ... }:

Este seletor h1 aplica estilos ao elemento de nível de cabeçalho <h1>. Define margem, padding, alinhamento de texto, tamanho de fonte e cor do texto para os títulos principais.
nav ul { ... }:

Este seletor nav ul aplica estilos à lista não ordenada dentro da navegação. Remove os estilos padrão da lista, como marcadores de lista e preenchimento, e centraliza o texto dentro da navegação.
nav ul li { ... }:

Este seletor nav ul li aplica estilos aos itens de lista dentro da navegação. Define-os como display: inline para que fiquem lado a lado e adiciona um espaçamento à direita para separá-los.
nav ul li a { ... }:

Este seletor nav ul li a aplica estilos aos links dentro dos itens de lista da navegação. Define a cor do texto, remove o sublinhado do link e define uma transição suave de cor ao passar o mouse sobre o link.
nav ul li a:hover { ... }:

Este seletor nav ul li a:hover aplica estilos aos links da navegação quando o cursor do mouse passa sobre eles. Neste caso, muda a cor do texto para amarelo (#ffc107) para indicar visualmente que o link está sendo selecionado.
main { ... }:

Este seletor main aplica estilos à seção principal do conteúdo do site. Define o espaçamento interno e centraliza o texto dentro desta área.
footer { ... }:

Este seletor footer aplica estilos ao rodapé do site. Define a cor de fundo, cor do texto, alinhamento de texto, espaçamento interno e fixa o rodapé na parte inferior da página.
