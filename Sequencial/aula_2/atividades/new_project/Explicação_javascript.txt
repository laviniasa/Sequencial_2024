Isso significa que o código dentro dessa função será executado assim que o documento HTML for completamente carregado. Isso garante que o código JavaScript só seja executado após a estrutura HTML estar pronta, o que é importante para garantir que os elementos HTML que o código precisa já estejam disponíveis.
